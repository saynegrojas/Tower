## Tower-Finder

An App that allows wireless site acquisition teams to locate existing towers near a specified address.

## Getting Started

To get started, make sure to clone the repo.

## Prerequisites

- Clone repository

## Installing

1. Clone repository
2. Open Live Server from HTML file.
